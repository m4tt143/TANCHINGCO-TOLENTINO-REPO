# TANCHINGCO-TOLENTINO-REPO

**Programming 2 Finals Project — UPHSD Molino, CCS**  
**Group 20:** Tolentino & Tanchingco

This repository contains the final project for Sir Val's Programming 2 course: **LAST OUTPOST: Last Stand**, a Survival Defense game built in Java.

## 🎮 Project Overview

**LAST OUTPOST: Last Stand** is a real-time survival defense game where you, as a lone soldier, must defend against endless waves of enemies. Featuring auto-targeting mechanics, strategic upgrade systems, and dynamic difficulty progression.

### Key Information
- **Language:** Java 17+
- **Framework:** Java Swing (AWT)
- **Platform:** Cross-platform (Windows, macOS, Linux)
- **Status:** Complete ✓

## 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/TANCHINGCO-TOLENTINO-REPO.git
   cd TANCHINGCO-TOLENTINO-REPO/LoneSoldier
   ```

2. **Compile:**
   ```bash
   javac *.java
   ```

3. **Run:**
   ```bash
   java Main
   ```
